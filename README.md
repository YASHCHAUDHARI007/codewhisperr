# ⚡ CodeWhisperr

> Understand any codebase in seconds.
> Powered by ultra-fast AI.

---

## 🚀 Overview

CodeWhisperr is a modern AI-powered web app that analyzes codebases and generates clear, structured explanations instantly.
Built for developers, students, and creators who want to **understand complex code without wasting time**.

---

## ✨ Features

* ⚡ Ultra-fast AI responses (Groq-powered)
* 🧠 Smart codebase explanations
* 💻 Clean & modern UI (dark + glassmorphism)
* 📂 Supports raw code or GitHub input
* 🚀 Optimized for speed and scalability

---

## 🖼️ UI Highlights

* 🌌 Dark futuristic design
* 💎 Glassmorphism cards
* ⚡ Smooth animations
* 🎯 Minimal & distraction-free

---

## 🛠️ Tech Stack

* **Frontend:** Next.js 15 + React
* **Styling:** Tailwind CSS
* **AI Backend:** Groq (LLaMA 3)
* **Deployment:** Netlify

---

## ⚙️ How It Works

1. Paste your code or repository
2. Click **Analyze**
3. AI processes and explains the structure
4. Get a clean, readable breakdown

---

## 📦 Installation

```bash
git clone https://github.com/YASHCHAUDHARI007/codewhisperr.git
cd codewhisperr
npm install
npm run dev
```
